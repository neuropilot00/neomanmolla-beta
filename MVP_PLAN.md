# 너만 몰라 MVP 기획

## 한 줄 정의

4명이 2분 안에 즐기는 모바일 소셜 추리 게임. 한 명만 다른 질문을 받고, 답변의 어색함을 보고 가짜를 찾는다.

## 타깃

- 친구끼리 가볍게 할 파티게임을 찾는 10대 후반-20대
- 방송 중 시청자와 같이 추리할 수 있는 스트리머
- 동아리, MT, 워크숍에서 아이스브레이킹 게임이 필요한 진행자

## 핵심 가설

1. 10초 안에 룰을 이해하면 초대 전환이 오른다.
2. 답변 공개 후 "왜 그렇게 답했어?"가 나오면 재미가 발생한다.
3. 결과 화면이 공유/클립용이면 자연 유입이 생긴다.
4. 과금은 일반 플레이어가 아니라 방장, 스트리머, 행사 진행자에게 붙인다.

## 베타 범위

### 반드시 있어야 하는 것

- 혼자 플레이: 봇 3명과 가짜 찾기
- 친구 데모: 4-6명 흐름을 한 기기에서 체험
- 방 만들기: 테마와 질문팩 선택
- 질문팩 3종 이상: 일상, 연애/썸, MT/술자리
- 결과 화면: 정답, 가짜, 점수, 공유 문구
- 베타 피드백 화면: 룰 이해, 납득, 재초대 의향 검증
- 픽셀아트 키비주얼과 캐릭터 아바타

### 이번 프로토타입에서 보여줄 것

- 핵심 루프: 시작 -> 답변 보기 -> 투표 -> 결과 -> 다음 문제
- 수익화 미리보기: 질문팩, 결과 카드, 방장 Pro, 스트리머 모드
- MVP 지표 카드: 초대율, 재시도율, 공유율

### 다음 개발 단계

- 링크 기반 방 생성
- Supabase/Firebase 기반 실시간 동기화
- 플레이어별 답변 입력
- 질문팩 관리
- 결과 카드 이미지 생성
- 스트리머 관전 링크와 시청자 예측 투표

## 수익화

- 무료: 혼자 플레이, 기본 질문팩, 4인 방
- Plus: 광고 제거, 질문팩 확장, 프로필 프레임, 커스텀 질문
- Pro: 관전 링크, 시청자 투표, 방 테마, 결과 카드
- Event: 행사 전용 방, 로고/테마, 진행자 기능

## 검증 지표

- 첫 플레이 완료율
- 3판 이상 재시도율
- 친구 초대 클릭률
- 결과 공유 클릭률
- 질문팩 구매/Pro 관심 클릭률
