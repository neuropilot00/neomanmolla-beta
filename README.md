# 너만 몰라

혼자서도, 4~6명이서도 짧게 즐기는 모바일용 소셜 추리 게임 베타입니다.

한 명만 다른 질문을 받고, 모두의 답변을 공개한 뒤 누가 어색한지 찾아냅니다.

## 실행

`index.html`을 브라우저로 열면 바로 실행됩니다.

## 현재 구현

- 데이터/콘텐츠 분리: `game-data.js`
- 4명, 5명, 6명 방 인원 선택
- 혼자 플레이 모드
- 내 프로필 꾸미기
- 방 만들기 설정
- 4인 픽셀 캐릭터 기반 봇 답변 가짜 찾기
- 점수, 콤보, 라이프
- 힌트 사용
- 픽셀아트 메인 비주얼
- 정면 픽셀 캐릭터 아바타 6종
- 라운드별 질문 카드
- 한 명만 다른 질문을 받는 역할 공개 흐름
- 자동 답변 공개
- 토론 화면
- 투표와 결과 화면
- 결과 카드와 베타 피드백 화면
- 모바일 우선 UI

## 다음 단계

- 실제 방 코드와 실시간 동기화
- 봇별 성격과 말투 강화
- 플레이어별 답변 입력
- 타이머와 채팅/음성 연동
- 스트리머 관전 링크와 시청자 예측 투표
