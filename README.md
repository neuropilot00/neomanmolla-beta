# 너만 몰라

혼자서도, 4~6명이서도 짧게 즐기는 모바일용 소셜 추리 게임 베타입니다.

한 명만 다른 질문을 받고, 모두의 답변을 공개한 뒤 누가 어색한지 찾아냅니다.

## 실행

`index.html`을 브라우저로 열면 바로 실행됩니다.

## 현재 구현

- 데이터/콘텐츠 분리: `game-data.js`
- 한국어/일본어 언어 전환
- 일본어 질문팩/테마/결과 문구
- 플레이어별 한국어/일본어 질문 표시
- 케이팝 팬덤 질문팩
- BTS, SEVENTEEN, Stray Kids, ENHYPEN, TXT, BLACKPINK, TWICE, aespa, IVE, LE SSERAFIM 내부 테스트용 그룹 데이터
- 최애 그룹/포지션풍 프로필 꾸미기
- 전신 캐릭터 꾸미기: 헤어, 의상, 소품, 오라, 프레임
- 확장 가능한 언어 선택 UI
- URL 기반 테스트 방 링크
- 로컬 설정 저장
- 4명, 5명, 6명 방 인원 선택
- 플레이어 이름 입력
- 혼자 플레이 모드
- 내 프로필 꾸미기
- 방 만들기 설정
- 4인 픽셀 캐릭터 기반 봇 답변 가짜 찾기
- 점수, 콤보, 라이프
- 힌트 사용
- 픽셀아트 메인 비주얼
- 정면 픽셀 캐릭터 아바타 6종
- 라운드별 질문 카드
- 한 명만 다른 질문을 받는 역할 공개 흐름
- 플레이어별 직접 답변 입력
- 테스트용 자동 답변 입력
- 토론 화면
- 투표와 결과 화면
- 결과 카드와 베타 피드백 화면
- 결과/초대/피드백 복사
- 런칭 전 검증 체크리스트
- 모바일 우선 UI

## 베타 테스트 방법

1. `방 만들기`에서 질문팩과 테마를 선택합니다.
2. `초대 링크 복사`로 테스트 유저에게 링크를 보냅니다.
3. 링크를 받은 유저는 방 참가 화면에서 로컬 데모를 실행합니다.
4. 플레이 후 `결과 복사` 또는 `베타 피드백 보기`로 의견을 회수합니다.

일본어 테스트는 첫 화면에서 `日本語`를 선택하거나 링크에 `lang=ja`를 붙이면 됩니다. 한일 혼합 테스트는 방 준비 화면에서 플레이어별 언어를 선택하면 같은 라운드가 각자 언어로 표시됩니다.

K-pop 그룹 데이터는 내부 테스트용 큐레이션입니다. 정식 공개 전에는 권리/표기/업데이트 정책을 별도 확정해야 합니다.

현재 버전은 정적 베타라 실시간 멀티 동기화는 없습니다. 실제 방 동기화는 다음 단계에서 Supabase 또는 Firebase로 연결합니다.

## 오픈 직전 기준

- 첫 화면에서 룰을 10초 안에 이해할 수 있어야 합니다.
- 결과 화면에서 공통 질문과 가짜 질문의 차이가 바로 보여야 합니다.
- 최소 3판 이상 반복 플레이 의향을 확인해야 합니다.
- 초대 링크를 실제 친구에게 보낼 만큼 민망하지 않아야 합니다.
- 질문팩 1개 이상은 유료로 팔 수 있을 만큼 반응이 강해야 합니다.
